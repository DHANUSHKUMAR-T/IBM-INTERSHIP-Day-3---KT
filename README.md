# Student Management System  
### IBM Naan Mudhalvan Internship Project

## Project Overview
The Student Management System is a full-stack web application developed as part of the IBM Naan Mudhalvan Internship Program.  
It allows users to manage student records using a React frontend and a Node.js & Express backend.

This project demonstrates the implementation of RESTful APIs and basic CRUD operations.

## How to Run the Project

### Step 1: Clone the Repository
git clone <your-github-repository-link>  
cd IBM-Internship-Student-Management-System

### Step 2: Run Backend Server
cd backend  
npm install  
npm start  

Backend runs on:  
http://localhost:5000

### Step 3: Run Frontend Application
cd frontend  
npm install  
npm start  

Frontend runs on:  
http://localhost:3000

---

## API Endpoints

Method | Endpoint | Description
-------|----------|-------------
GET | /api/students | Fetch all students
POST | /api/students | Add a new student
PUT | /api/students/:id | Update student details
DELETE | /api/students/:id | Delete a student
